メッセージハウスは、データサイエンスにおける価値や差別化要因を記述・議論するための構造を提供します。

このユースケースについては、現時点ではポジショニングステートメントを設けていません。一般的に、ポジショニングステートメントには「このユースケースの市場において GitLab がどのように位置付けられ、どのように差別化されるか」を記述します。

| **キーバリュー** | バリュー 1: *<キーメッセージ／バリュープロポジションを記載>* | バリュー 2: | バリュー 3: |
|--------------|------------------------------------------------------------------|----------|----------|
| **約束（Promise）** | *(ポジティブなビジネス成果を列挙し、説明する)* |  |  |
| **ペインポイント** | *(共通のペインポイントを記述する)* |  |  |
| **GitLab を選ぶ理由** | *(このバリューを支える具体的な機能を列挙する)* |  |  |

| **証跡（Proof points）** | *(具体的なアナリストレポート、ケーススタディ、お客様の声などを列挙する)*  |
