## マージリクエストコーチへの応募

- `mr_coach_onboarding` テンプレートを使って [新しい Issue](https://gitlab.com/gitlab-org/developer-relations/contributor-success/team-task/-/issues/new?issue_template=mr_coach_onboarding) を作成します（下のスクリーンショット参照）。
- Issue テンプレートのプレースホルダーをご自身の情報で埋めます。
- その Issue を自分自身に割り当てます。
- 新しく作成した Issue に書かれた手順を進めていきます。

![mr_coach_onboarding の Issue テンプレートを選択する方法を示す新規 Issue ダイアログ](/images/marketing/developer-relations/contributor-success/new_mr_coach_onboarding_issue.png)

## 円満に役割から退く方法

マージリクエストコーチを続けられなくなった場合は、残るコーチたちのキャパシティが変わらないように、自分の代わりとなる別の GitLab チームメンバーを見つけてください。退く準備が整ったら、次の作業を行います。

1. `#mr-coaching` の Slack チャンネルでアナウンスします。
1. `team.yml` ファイルを更新し、「Merge Request Coach」を削除します。
1. `@gitlab-org/coaches` グループから自分を外します。
