各スタジオはプロダクト領域に対応するステージグループの名前付きのまとまりであり、それらのグループ全体にわたる業務を担当するシニアデザインリーダーが配置されています。

## スタジオ

- Security
  - 対象: Security と Software Supply Chain Security
  - デザインマネージャー: Marissa Henri
- Core DevOps + Data
  - 対象: Create、Plan、Verify、Package、Data
  - デザインマネージャー: Gina Thisse
- Artificial Intelligence
  - 対象: AI 搭載機能と Duo プロダクトスイート
  - デザインマネージャー: Alex Kozovski
- PLG/Monetization/Platform
  - 対象: Growth、Fulfillment、Govern、プラットフォーム全体のプロダクト領域
  - デザインマネージャー: Emily Sybrant
