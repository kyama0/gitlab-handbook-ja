## Slack チャネルとグループ

フィールドの主要な Slack チャネルは次のとおりです。

1. #field-fyi: 公式のフィールドアナウンスチャネル。重要な更新を効率化するために権限レベルが制限されています。(詳細は [こちら](/handbook/sales/sales-google-groups/field-fyi-channel)。)
2. #field-managers: フィールド組織のすべてのマネージャー以上の公式チャネル。
3. #sales: WW Sales の一般チャネルで、すべてのチームメンバーからの投稿に開かれています。

すべてのフィールドチームメンバーは、ビジネスに関する重要な更新のためにこれら 3 つのチャネルを[スター](https://slack.com/help/articles/360043207674-Organize-your-sidebar-with-custom-sections) 🌟 してフォローすべきです。フィールド組織には、以下にアウトラインされている複数のチームチャネルがあります。役割に関連するチャネルをフォローしてください。プライベートチャネルへのアクセスを得るには、[アクセスリクエスト (AR) フォームに記入してください](/handbook/security/corporate/end-user-services/access-requests/)。

### Slack グループ

Slack グループとは、ワークスペース内のメンバーのグループで、しばしば一度に通知される必要があります。フィールド組織には、Slack コミュニケーションで人々のグループに効果的に CC できる複数の Slack グループがあります。Slack グループはすべてのメンバーにプッシュ通知を送るので、[ハンドブックの @here メンションに関するガイドライン](/handbook/communication/#be-respectful-of-others-time)に従って、控えめに使用するべきです。

Slack グループは以下の場合にのみ使用してください。

1. メッセージのコンテンツが、グループのすべてのメンバーに極めて関連性が高い
2. メッセージのコンテンツが時間に敏感である、または即座のアクションが必要
3. メッセージのコンテンツが、ビジネスおよび/またはチームに重大な影響を与える

| グループ名 | グループメンバー/説明 |
| ------ | ------ |
| field_managers | ASM 以上 *または* CS/Channel/Alliance/SDR 側で同等のピープルマネージャー |
| sales_leaders | Regional director 以上 (COM の場合は ASM 以上) |
| cs_leaders | VP Customer Success の直属の部下 |
| sales_ent_managers | Enterprise チームの ASM 以上 |
| sales_com_managers | Commercial チームの ASM 以上 |
| sales_cs_managers | Customer Success チームのマネージャー以上 |
| sales_dev_leadership | Sales Dev チームのマネージャー以上 |
| salesdev_global | すべての Sales Dev チームメンバー |

[公開チャネルを使用](/handbook/communication/#use-public-channels)するという企業のガイダンスと一致して、プライベートチャネルが好まれる場合があります。[包摂的な環境](/handbook/values/#diversity-inclusion)を確保したいと考えているため、一部のチームのフィードバックは、判断されているという認識 (すなわち、インポスターシンドローム) があるため、プライベートチャネルが時々好まれるということです。推奨は、公開チャネルを使用することです。なぜなら、それは:

- ソリューション、推奨事項、ベストプラクティスについて、より広いコラボレーション (すなわち、SA/CSM/PS、Product、Marketing、Sales) を可能にする
- 顧客の課題、イネーブルメントのニーズ、製品ドキュメントの改善など、より広い学習の能力を提供する
- コンテンツ (すなわち、CSM ソリューション、デジタルジャーニーのアダプションコンテンツ、製品ドキュメント) の情報を収穫する能力を可能にする
推奨は、時々不快に感じるかもしれない場合でも公開チャネルを使用することです。なぜなら、私たちは[これを受け入れたいと思っている](/handbook/values/#embracing-uncomfortable-ideas-and-conversations)からです。
