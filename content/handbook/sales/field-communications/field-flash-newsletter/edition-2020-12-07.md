### What's New in GitLab 🚀

GitLab 13.6 リリースが公開されました！ 顧客および見込み客向けの要点は以下のとおりです:

- **セキュリティとコンプライアンスのリスク低減:** いくつかのダッシュボードとレポートの改善により、チームはより速い意思決定を可能にする可視性を得られます。これには、マージ前に解決すべき重要なコード品質違反を素早く判断するのに役立つ Full Code Quality Report、リアルタイムおよび過去の脆弱性トレンドを把握し続けるのに役立つ Project Security Dashboard の更新、ユーザー、プロジェクト、グループ、Issue、パイプラインなどの人気機能の過去 12 か月間の使用傾向を確認できる GitLab Self-Managed 管理者向け機能などが含まれます。
- **運用効率の向上:** 人気のあるツールとの拡張性が向上し、ツール間で簡単にアクセス・コラボレーションできます。これには、Snippet の挿入、マージリクエストや Issue の閲覧・コメントを VS Code から直接行えるよう強化された VS Code 拡張機能が含まれます。さらに、GitLab インテグレーションは、インスタンスおよびプロジェクトレベルに加えて、グループレベルでも設定できるようになり、グループオーナーがインテグレーションを簡単に管理できるようになりました。
- **より優れた製品をより速く提供:** Auto DevOps が AWS をサポートするように拡張されたため、Amazon Web Services (AWS) を使用した GitLab CI/CD の開始がこれまで以上に簡単になりました。チームは Kubernetes を使用せずに Auto DevOps を使って Amazon EC2 に Auto-Deploy できます。グループオーナーは、master ブランチではなく、新しいリポジトリ用のカスタムデフォルト初期ブランチ名を設定する柔軟性を得られるようになりました。

[詳細は 13.6 リリースポストをご覧ください](https://about.gitlab.com/releases/2020/11/22/gitlab-13-6-released/)

13.7 で何が予定されているかは、[キックオフ動画](https://www.youtube.com/watch?v=9cjBjJ4BE1M) を視聴するか、[キックオフレビューページ](https://about.gitlab.com/direction/kickoff/) をご覧ください。

### Team Member Spotlight 🔦

#### Timm Ideker - Area Sales Manager, ENT West (Global Director, Mid-Market Sales)

[Timm Ideker - Area Sales Manager, ENT West (Global Director, Mid-Market Sales, 2021-02-01 から有効)](https://gitlab.com/tideker)

![Timm-Ideker-TMS-Thumb](/images/sales/field-communications/field-flash-newsletter/TimmIdeker_TMS_thumb.png)

**どこで育ち、現在どこを拠点にしていますか？**

サウスダコタ州 Sioux Falls、アメリカで育ち、Kansas と Maine での短期間の滞在を経て、1996 年頃ミネソタに定住しました。Land of 10,000 Lakes を離れる予定はありません！

**お仕事で一番好きなところは？**

チームメンバーとの 1:1 のつながりが、私を満たしてくれます。

**GitLab の Values の中で、最も共感するものは？**

Everyone can contribute は mantra でもあり call to action でもあります。誰もが貢献しなければなりません。騎兵隊はおらず、*私たちが*クライアントの最も困難な課題のいくつかを解決する専門家なのです。

**GitLab で働く上で共有したい役立つヒントはありますか？**

[donut-be-strangers](/handbook/company/culture/all-remote/informal-communication/#the-donut-bot) は、あなたを GitLab のチームメンバーとつなげる素晴らしいテクノロジーです。私はここに来てから 1〜2 週間ごとにこれらのランダムなコーヒーチャットをしており、いつも別の人や私たちのビジネスについて新しいことを学んでいます。強くお勧めします！
