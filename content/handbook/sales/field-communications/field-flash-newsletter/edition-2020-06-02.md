### What's New in GitLab 🚀

GitLab 13.0 マイルストーンリリースが公開されました！[この動画](https://www.youtube.com/watch?v=dStIhTP1ckg&feature=youtu.be)で Scott Williamson が 13.0 リリースで期待できるテーマを解説しています。顧客および見込み客向けの要点は以下のとおりです:

- **セキュリティとコンプライアンスのリスク低減:** 改善された Application Security Testing 機能を活用すれば、セキュリティ脆弱性をより早期に発見・修正できます。（関連情報として、2020 年 [Gartner Magic Quadrant for Application Security Testing](https://page.gitlab.com/resources-report-gartner-magic-quadrant-ast.html) で Niche Player に選出されました。）また、GitLab における脆弱性オブジェクトのエンドツーエンドの取り扱いを改善し、脆弱性管理の効率を高めています。
- **運用効率の向上:** GitLab 13.0 は、ビジネスが既存のソフトウェア開発プロセスを合理化し、より迅速で効率的になるための機能に焦点を当てています。Amazon ECS へのデプロイの簡素化や、複数のソースから発生する IT アラートを集約する単一インターフェイスを提供する統合アラートリストなどが含まれます。
- **より優れた製品をより速く提供:** ロードマップ上でエピックの階層を表示でき、エピックがマイルストーンとどう整合しているかを可視化できるようになりました。これにより、複雑なプロジェクトの管理とチーム全体での迅速なコラボレーション・対応が可能になります。また、強化された Gitaly クラスターによる高可用性 Git ストレージで、重要な Git リポジトリの安全性も向上しています。

[詳細は 13.0 リリースポストをご覧ください](https://about.gitlab.com/releases/2020/05/22/gitlab-13-0-released/)

13.1 で何が予定されているかは、[キックオフ動画](https://www.youtube.com/playlist?list=PL05JrBw4t0KrgMRzgXIGTalGc6tzeoprn) を視聴するか、[キックオフレビューページ](https://about.gitlab.com/direction/kickoff/) をご覧ください。
