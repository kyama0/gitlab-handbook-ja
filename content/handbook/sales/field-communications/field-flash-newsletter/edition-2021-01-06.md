### What's New in GitLab 🚀

GitLab 13.7 リリースが公開されました！ 顧客および見込み客向けの要点は以下のとおりです:

- **運用効率の向上:** 13.7 はパッケージと依存関係管理への複数の機能強化を備え、開発時間の無駄を減らします。これには、Generic Package を素早く見つけて表示する追加オプションが含まれます。また、GitLab の Dependency Proxy への複数の改善も含まれます — Dependency Proxy で Docker レート制限を回避してパイプラインを高速化する機能、プライベートプロジェクトで Dependency Proxy を使用する機能、gitlab.ci-yml ファイル内で定義された変数や値のハードコーディングに頼る代わりに、Dependency Proxy で事前定義済み変数を使用する機能。
- **運用効率の向上:** 強化されたクロスコラボレーションにより、お客様はプロジェクトの管理に費やす時間を減らし、貴重な機能の開発により多くの時間をかけられます。13.7 の新しいプロジェクト管理機能により、グループとプロジェクト間で効率的にコラボレーションしやすくなりました。これには、merge request reviewers の追加が含まれ、お客様はマージリクエストに誰が関与しているかを素早く把握したり、通知を送信する正式なレビューをリクエストしたりできます。これには、クイックアクションで Issue をクローンする機能や、ブロッカーで Issue をソートして他の Issue の進捗をブロックしている Issue を明らかにする機能も含まれます。
- **より優れた製品をより速く提供:** 改善された自動化とデプロイメントの柔軟性により、価値をお客様の手にもっと早く、もっと頻繁に届けられます。これには、失敗したデプロイメントを最後の成功したデプロイメントに自動的に戻し、自動的に状態通知を送信する新機能や、Environment ページでデプロイメント状態を表示する機能が含まれます。13.7 は、Red Hat OpenShift 上の GitLab Runner コンテナの最初の公式サポートベータと、Certified Runner Operator も備えており、GitLab でのリリース方法にさらなる柔軟性を提供します。

[詳細は 13.7 リリースポストをご覧ください](https://about.gitlab.com/releases/2020/12/22/gitlab-13-7-released/)

13.8 で何が予定されているかは、[キックオフ動画](https://www.youtube.com/watch?v=bWKdW-s9o9E) を視聴するか、[キックオフレビューページ](https://about.gitlab.com/direction/kickoff/) をご覧ください。

### Team Member Spotlight 🔦

#### Ricardo Amarilla - Customer Success Manager

[Ricardo Amarilla - Customer Success Manager](https://gitlab.com/ricardo.amarilla)

![Ricardo-Amarilla-TMS-Thumb](/images/sales/field-communications/field-flash-newsletter/Ricardo_Amarilla_TMS_Thumb.png)

**どこで育ち、現在どこを拠点にしていますか？**

私はパラグアイのアスンシオンで生まれ育ち、現在もそこを拠点としています。海外に住んだことは一度もありません。

**お仕事で一番好きなところは？**

仕事をするのに最適なタイミングを選ぶ自由は、これまで経験したことのないものです。GitLab のおかげで、家族、友人、仕事の間の適切なバランスを見つけられます。

**GitLab の Values の中で、最も共感するものは？**

Collaboration です — GitLab では、誰もが助け合い、コラボレーションする意欲があり、これはユニークです。私たちは全員リモートですが、このレベルのコラボレーションのおかげで、孤立を感じることはありません。

**GitLab で働く上で共有したい役立つヒントはありますか？**

お客様は通常、MeetUp、LinkedIn、その他のプラットフォームに技術コミュニティを持っています。ネットワーキングのためにそれらのいくつかに参加することをお勧めします。新しいことを学ぶ以外に、いくつかのアカウントとの再エンゲージにも役立ちました。

**チームが知らないかもしれないあなたの趣味やスキルは？**

数週間前にパドルを始めました — 私にとって新しい趣味です。これ以外には、家族と過ごしたり、ランニングをしたり、自転車に乗ったりして余暇を過ごしています。

**アイスブレイカー: バケットリストで一番クレイジーなものは？**

アイアンマン・トライアスロンに参加すること。
