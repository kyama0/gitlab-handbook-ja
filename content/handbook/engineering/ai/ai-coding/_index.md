## 概要

AI Coding Group は、開発者がコードを書き、理解し、改善するのを支援する AI を活用した機能を開発しています。このグループは機能別のチームに編成されており、それぞれが特定の領域に注力しています。

## 機能別チーム

1. [Code Review](code-review/) - Code Review Workflow と Duo Code Review の機能を横断するコードレビュー組織。
1. [DAP Events](dap-events/) - Duo Flows を自動的に実行できるようにするイベントプラットフォームおよびトリガーシステム。
1. [DAP Repository Flows](dap-repository-flows/) - リポジトリを健全かつ最新の状態に保つ DAP オンボーディングとバックグラウンドエージェント。

## Feature Stewardship

- [Feature Stewardship](feature-stewardship/) - 恒久的なオーナーシップが決まるまでの間、AI Coding の管理下にある機能。Code Suggestions、AI Assisted Service、Duo Context Exclusion を含みます。

## 私たちの働き方

オンボーディング、休暇、Issue ボード、ミーティングなど、チームの働き方に関する情報については [How We Work](how-we-work/) ページを参照してください。
