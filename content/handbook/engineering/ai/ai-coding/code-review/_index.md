## 概要

AI Coding 内の Code Review 組織は、AI を通じてコードレビューをより高速で高品質にすることに注力しています。この組織は、中核となるコードレビューワークフローと AI を活用した Code Review 機能の両方にまたがっており、統一されたチーム構造に向けて取り組んでいます。

この組織には現在、2 つのファンクショナルチームがあります:

1. [Code Review Workflow](workflow/) - 中核となるコードレビューワークフロー機能。Create ステージから AI Coding に移行中です。
1. [DAP Code Review](dap/) - マージリクエスト向けの AI を活用した DAP Code Review 機能。
