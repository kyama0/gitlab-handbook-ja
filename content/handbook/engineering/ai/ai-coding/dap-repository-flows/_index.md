## Overview

DAP Repository Flows チームは、関連する 2 つの領域に取り組んでいます。

**DAP Onboarding**: プロジェクトをよりエージェント的にするファイルと設定を作成します。リポジトリを AI 対応にし、Duo 機能がその中で効果的に動作できるよう支援します。

**Background Agents / Repository Flows**: リポジトリをバックグラウンドで実行され、時間をかけて積極的に改善する自律フローです。不足しているテストの発見、ドキュメントの作成、コードの最新化などを行います。これらのフローは手動での呼び出しを必要とせずに実行され、リポジトリを継続的に改善します。
