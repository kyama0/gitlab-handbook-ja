## 概要

DAP Repository Flows チームは、関連する 2 つの領域に取り組んでいます:

**DAP オンボーディング**: プロジェクトをよりエージェント的に活用しやすくするファイルと設定を作成します。これにより、リポジトリを AI 対応にして、その中で Duo の機能が効果的に動作できるよう支援します。

**Background Agents / Repository Flows**: リポジトリに対してバックグラウンドで実行され、時間の経過とともにプロアクティブに改善する自律的なフローです。テストの不足を見つける、ドキュメントを書く、コードを最新の状態に保つ、などを行います。これらのフローは手動で呼び出す必要なく実行され、リポジトリを継続的に改善します。
