## 概要

AI Core Infra グループは、Duo Workflow Service と AI gateway のインフラストラクチャおよび信頼性改善、すべての AI 機能が依存するモデル評価ツール、ならびに Semantic Code Search を所有しています。

1. [Duo Service Infra](duo-service-infra/) - セルフホスト型 AIGW、および AI Gateway と Duo Workflow Service の可観測性と信頼性の改善
