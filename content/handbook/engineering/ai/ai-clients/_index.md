## 🚀 ミッション

AI Clients 組織は、顧客が GitLab Duo を体験するクライアント側の接点を担います。顧客がすでに使用しているツールに GitLab の AI 機能を直接提供し、プラットフォームをまたいでそれらの体験を支える共有クライアント SDK を提供します。

---

## 💬 連絡先

### Slack

私たちは、[GitLab Slack チャンネルのプレフィックス規約](/handbook/communication/#slack)に従います。`s_` はチーム横断のステージ / 組織チャンネル、`f_` は機能別チームのチャンネル（エンジニアリング、PM、UX の合同）、`g_` はエンジニアリングのみのグループディスカッションを示します。アンダースコアは異なる要素を区切り、ハイフンは単一の要素内の単語をつなぎます。

| 範囲 | チャンネル | 目的 |
| ----- | ------- | ------- |
| 組織 | `#s_ai-clients` | すべての職種（エンジニアリング、PM、UX、ステークホルダー）に向けた、チーム横断の AI Clients ディスカッション |
| Duo Client SDK | `#f_language_server` | Duo Client SDK / Language Server の機能別チーム |
| Duo CLI | `#f_duo-cli` | Duo CLI の機能別チーム |
| VS Code | `#f_vscode_extension` | VS Code 拡張機能の機能別チーム |
| JetBrains | `#f_jetbrains_plugin` | JetBrains プラグインの機能別チーム |

エンジニアリングのみのグループチャンネル（`#g_duo_chat`、`#g_editor-extensions`）は、メンバーとステークホルダーが上記の `f_` / `s_` チャンネルに移行する間、当面は存続します。
