## ビジョン

VS Code チームは Visual Studio Code 向けの GitLab 拡張機能を担当し、GitLab の中核機能と AI 機能をエディターに提供します。機能開発の大部分は共有 Language Server (`gitlab-lsp`) 上に構築されるため、クライアント間で再利用できます。
