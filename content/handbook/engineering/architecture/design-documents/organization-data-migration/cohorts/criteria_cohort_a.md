詳細な対象条件が確定するまでのプレースホルダードキュメントです。

| 属性 | 値 |
| --- | --- |
| Cohort 名 | 非アクティブな Free ユーザーのサブセット |
| 対象規模 | 最大 5,000 organizations |
| 公開範囲 | Private のみ |
| エンゲージメントモデル | なし |
| 前提条件 | Cohort 0 において、移行プロセスが機能することが検証済みであること。 |
| 想定される影響 | データベースサイズへのごくわずかな影響 |
