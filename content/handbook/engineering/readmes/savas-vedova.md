## Savas Vedova の README

こんにちは、

私の名前は Savas（サヴァシュと読みます）で、GitLab の Govern チームの Senior Frontend Engineer です。このページは私自身と私の仕事のやり方についての簡単な README です。

人生の半分をスイスで、もう半分をトルコで過ごしたため、自分をバイカルチャーな人間だと思っています。これにより、友達をすぐに作り、他の文化にもすぐに馴染めるようになりました。英語、イタリア語、トルコ語の3言語を流暢に話せます。フランス語の知識もあり、ドイツ語は税務署で用事を済ませられる程度には話せます。

ランニング、水泳、友人との外出、家族との時間を楽しんでいます。自然とそのすべての美しさが大好きです。

## 私の仕事スタイル

* ほぼいつでも対応可能です。仕事が好きです。
* タスクを任されたら、期待される納期内に完了するよう全力を尽くします。
* 習得が早い方です。

## 他者に対して想定すること

* 人はそれぞれ異なるので、相手の立場に立って行動するよう心がけています。
* 話している相手がどれほど経験豊富であっても、コンテキストがないと仮定して、最も明白なことでも説明しようとします。これは相手に知識がないと思っているからではなく、質問を恥ずかしいと感じる人（自分自身を含む）がいるので、誰もその立場に立たせたくないからです。

## 私とのコミュニケーション

* とても気さくです — できるだけ直接的に話しかけてください。
* 非同期コミュニケーションを好みますが、非同期がうまく機能しない場合はほぼいつでも同期コミュニケーションに対応できます。
