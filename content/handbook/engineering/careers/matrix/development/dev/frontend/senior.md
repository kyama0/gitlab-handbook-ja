### テクニカルコンピテンシー {#technical-competencies}


<!-- include omitted: includes/engineering-careers/senior-technical-competency.md (no localized version under content/ja/) -->


<!-- include omitted: includes/engineering-careers/development-senior-technical-competency.md (no localized version under content/ja/) -->


- フロントエンドプログラミング言語と GitLab のデフォルトフロントエンドフレームワークに高度な専門知識を持つ
- REST および GraphQL API 設計に関するアーキテクチャ上の指針を提供できる
- WebSocket を活用してリアルタイム更新 UI を構築できる
- ドキュメントの品質にかかわらず、サードパーティライブラリと連携できる
- GitLab のコードベースにおける低速・非効率なフロントエンド実装を特定し対処する
- サービスや技術スタックの複数レベルにまたがるプロダクション問題をデバッグする
