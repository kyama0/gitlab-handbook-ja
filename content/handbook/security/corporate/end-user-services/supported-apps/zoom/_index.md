Zoom は私たちの主要な音声・映像コミュニケーションソフトウェアで、社内および外部パートナーとのコミュニケーションに使用されています。シンプルなビデオ通話に加え、自動録画や共有ホワイトボードなど多くの追加機能も備えています。

## プライバシーとセキュリティに関する注意事項

Zoom アプリは EUS が Jamf を通じて直接管理しています。これは、機能とセキュリティの両面でアプリを最新の状態に保つため、私たちが定期的にアップデートを自動でプッシュしていることを意味します。

私たちはまた、新機能を GitLab 内で使用可能にする前に、セキュリティリスクの観点からレビューします。その結果、最新機能を使用できるようになるまでに遅延が発生したり、私たちが完全にブロックすることを決定したりする場合があります。

Zoom を使用する際は、以下を必ず実施してください。

- ノートパソコンの OS と Zoom アプリを最新の状態に保つこと。アップデートが利用可能になると、手順とともにデスクトップ通知を受け取ります。**アップデートしていない場合、アプリを使用できない可能性があることにご注意ください**
- 無制限通話と高度な機能にフルアクセスするため、Zoom アプリに正しくサインインしていることを確認すること
- [ノートパソコン設定ガイドライン](/handbook/security/corporate/end-user-services/laptop-management/laptop-security) に従っていることを確認すること
- Zoom プロフィールを更新し、カスタマイズされた [パーソナルリンク](/handbook/security/corporate/end-user-services/supported-apps/zoom/zoom-additional-config/#create-a-customized-personal-link) を作成することを検討してください
