## 概要

**初期セットアップ時に Touch ID を有効にしてください。**

GitLab では 2FA に WebAuthn を使用しています。つまり、Okta やさまざまな Web アプリケーションへのサインイン時に、生体認証 (Face ID／Touch ID／指紋) またはセキュリティキー (YubiKey など) の入力を定期的に求められます。

- [ベンダードキュメント - Mac で Touch ID を使用する](https://support.apple.com/guide/mac-help/use-touch-id-mchl16fbf90a/14.0/mac/14.0)
- [ベンダードキュメント - Mac の Touch ID とパスワードの設定](https://support.apple.com/guide/mac-help/touch-id-password-settings-on-mac-mchl319754b3/14.0/mac/14.0)
