## 概要

私たちは、個別の雇用契約や合意に矛盾する記述がない範囲で、[内部利用規程 (Internal Acceptable Use Policy)](/handbook/people-group/acceptable-use-policy/) に従い、チームメンバーがラップトップを限定的な個人利用 (各自の判断による) に使用することを許可しています。これにはゲームや個人的なブラウジングも含まれます。ソーシャルメディア、オンラインバンキング、一般的な消費者向けアプリケーションも対象です。仕事用コンピューターを職場閲覧不適切 (NSFW) なウェブサイトに使用するのは決して良い考えではありません。

私たちが懸念するのは、ポップアップが多数表示される、クリックベイトがある、有名人の写真を複数ページにわたって掲載している、またはマルウェアの被害を受けやすいウェブサイトです。つまり、Facebook のニュースフィードを閲覧することはできますが、広告を開くのは避けてください。マルウェアのインストールに対する脆弱性がはるかに低い個人のスマートフォンやタブレットからこれらのウェブサイトにアクセスしてください。

オンボーディング Issue の一環として、[内部利用規程 (Internal Acceptable Use Policy)](/handbook/people-group/acceptable-use-policy/) を読むことが必須となります。

ご使用のコンピューターは [Jamf](/handbook/security/corporate/systems/jamf) (MDM) と [SentinelOne](/handbook/security/corporate/systems/sentinelone) (EDR) によって監視されており、**訪問したすべての URL も含まれる**ことに注意してください。これは事後的なセキュリティ調査のためにのみ使用され、私たちは積極的にあなたのアクティビティを監視しているわけではありません。つまり、「ビッグブラザー」型の監視アプローチではなく、コンピューターが侵害された場合に可能な限り多くのフォレンジック追跡データを保持するためのものです。私たちが閲覧できるデータについては、この[スライドデック](https://docs.google.com/presentation/d/1C2ufNXF28l0KTd5PPTkq1TjUWeWPI44VfwYbsvOzkns/edit)で詳細を確認できます。
