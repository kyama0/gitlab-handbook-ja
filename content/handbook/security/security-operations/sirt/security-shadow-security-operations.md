## セキュリティインシデントレスポンスチーム (SIRT)

GitLab のセキュリティインシデントレスポンスチームは、GitLab SaaS および組織としての GitLab の最初の防衛線です。SIRT の主な目標は、セキュリティインシデントによる損害を最小化し、コントロールすることです。これは、セキュリティインシデントが発生したときを特定するための検知ツールの開発と展開、その範囲と影響を制限するためのイベントの特定と封じ込めの行動、セキュリティイベントを引き起こした根本問題の是正、そして運用が正常に戻れるようにセキュリティイベントから復旧することによって達成されます。インシデント対応に加えて、SIRT は、サイバー防御ツールの管理と展開、私たちのインフラストラクチャとチームメンバーのセキュリティベストプラクティスへの準拠の確保、インシデントレスポンスプロセスに関する保守とトレーニングを通じて、プロアクティブにセキュリティインシデントを防止する活動を行います。
