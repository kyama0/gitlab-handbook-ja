## 新しいチームメンバーへの注意

ローテーション ([HackerOne](/handbook/security/product-security/psirt/runbooks/hackerone-process/) または [トリアージローテーション](/handbook/security/product-security/security-platforms-architecture/application-security/runbooks/triage-rotation/)) に参加しているとき、またはオンボーディングプロセスを進めていて助けやアドバイスが必要なときは、`#security_help` Slack チャンネルで連絡するか、AppSec Sync ミーティングで尋ねてください。質問や助けが必要になりそうなシナリオの例を以下に挙げます:

- オンボーディングタスク、脅威モデリング、または AppSec レビューを行っており、行き詰まっている、あるいは何かを特に対処する方法が分からない場合
- ピングローテーションに参加していて、特定の状況にどう対処するか、特定の質問にどう取り組むかが分からない場合
- HackerOne ローテーションに参加していて、難しいレポートに対処しなければならない場合
