Teleport は、ユーザーがさまざまなリソースへの読み取りまたは書き込みアクセスを取得できるようにするアクセス管理プラットフォームです。Teleport はインフラストラクチャセキュリティが所有しています。

[インフラストラクチャ Runbook ドキュメント](https://gitlab.com/gitlab-com/runbooks/-/tree/master/docs/teleport)には、Teleport の使用と相互作用に関するさまざまな詳細トピックが含まれています。

ヘルプが必要な場合:

- **Teleport へのアクセスを取得する** - [アクセスリクエストを開きます](https://gitlab.com/gitlab-com/team-member-epics/access-requests/-/issues/new?issuable_template=Individual_Bulk_Access_Request)。
- **本番リソース (SSH、Rails コンソール、データベースなど) へのアクセスを取得する** - [Teleport 経由でデータベースコンソールに接続する](https://gitlab.com/gitlab-com/runbooks/-/blob/master/docs/teleport/Connect_to_Database_Console_via_Teleport.md) または [Rails コンソールに接続する](https://gitlab.com/gitlab-com/runbooks/-/blob/master/docs/teleport/Connect_to_Rails_Console_via_Teleport.md) の Runbook に従ってください。
- **一般的な支援** - インフラストラクチャセキュリティに [Issue を開きます](https://gitlab.com/gitlab-com/gl-security/product-security/product-security-engagements/product-security-requests/-/issues/new?description_template=infrasec-teleport)。
