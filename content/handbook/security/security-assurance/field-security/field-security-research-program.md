## なぜリサーチプログラムを?

Field Security の重要なミッションの 1 つは、ソートリーダーシップを通じてより広いセキュリティ業界に影響を与えることです。DevSecOps のリーダーかつセキュリティ業界の主要プレーヤーとして、GitLab は内部のセキュリティ専門知識を活用してセキュリティ標準を改善することに取り組んでいます。

## どのように?

Field Security の目標は、Security アウトリーチに包括的なアプローチをとることです。私たちはダイナミックでありたいと考え、情報の収集と共有の伝統的な方法を超越したいと考えています。

### アウトリーチエンゲージメント

- セキュリティ業界全体からの Request for Comments / Information への回答
- 業界のセキュリティ Working Group への参加
- 業界団体や制度的セキュリティ機関とのコミュニケーション
- 顧客のセキュリティに影響を与える規制、標準、フレームワークの考察

## あなたはどのように協力できるか?

すべてのチームメンバーからの継続的でオープンなフィードバックは何より歓迎されます。見込み顧客や顧客と接していて、懸念事項、提案、「ウォンツ」、「ニーズ」を学んだ場合は、その情報を Field Security に転送してください。あなたのフィードバックは、私たちが GitLab がより多くの価値を顧客と業界に提供できるよう助けるために、私たちのロードマップと戦略を推進する力となります。
