## ナレッジベースとは？

ナレッジベースは、GitLab, Inc. および GitLab ソリューションに関連する Q&A ペアのライブラリです。ナレッジベースは、見込み客や顧客からの質問票によって更新・拡張される生きたライブラリです。これらの質問票は、RFx リクエスト、サードパーティリスクアセスメント、サプライヤーデューデリジェンスリクエストなどの取り組みに関連しています。ナレッジベースは、Field Security と他のすべてのチームメンバーの両方に役立つように設計されています。

## ナレッジベースの使い方は？

ナレッジベースは、他のデジタルや物理的なライブラリと同じように、**質問への回答や調査の実施に**使用できます。

ナレッジベースはセルフサービスツールであり、見込み客や顧客の関心事に関連するキーワードを素早く検索できます。検索すると、入力したキーワードに一致する Q&A ペアのリストが表示され、見込み客や顧客に自信を持って回答するために必要な情報が得られます。

### ナレッジベースのユースケースには何がある？

- 見込み客や顧客とのミーティングのトーキングポイントを準備する
- 見込み客や顧客からのメールに返信する
- GitLab ソリューションと、GitLab, Inc. が社内でどのように運営しているかについて学ぶ

### Field Security はナレッジベースをどのように使用するのか？

- [顧客向けアシュアランス活動](/handbook/security/security-assurance/field-security/customer-security-assessment-process/) を実行する際の効率と効果を高めるため

## ナレッジベースへのアクセス方法は？

ナレッジベースは、当社の Trust Center ツールである [SafeBase](https://app.safebase.io/knowledge) を利用して提供されています。

ナレッジベースのナビゲーション方法について詳しく知るには、こちらの[ページ](https://help.safebase.io/en/articles/6117207-knowledge-base-overview#h_615c8940e2)をご確認ください！

ナレッジベースの閲覧者アクセス権を持つと、**Slack** を通じて Q&A にアクセスすることもできます！ 任意の Slack チャンネルで **/kb-search *security keywords*** と入力するだけで、候補となる回答が表示されます。SafeBase でエントリを表示するためのリンクも利用できます。

もちろん、アクセスレベルにかかわらず、誰でも Trust Center から直接ナレッジベースを閲覧して照会できます。

## 保守と改善

ナレッジベースは継続的に更新・拡張されている生きたライブラリです。Field Security が質問票を完了するにつれて、各ライブラリは最新かつ正確な回答で更新されます。Field Security はナレッジベースの構造とタグ付けをアドホックに更新し、すべてのチームメンバーにとってナビゲートしやすく効果的なセルフサービスツールであり続けることを保証します。

改善のご提案がある場合は、以下に記載された連絡方法のいずれかで Field Security までご連絡ください。
