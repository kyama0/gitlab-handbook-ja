## Women in Security ミッションステートメント

GitLab の Women in Security グループは、GitLab セキュリティ部門の女性のサポート、スキルおよびリーダーシップ開発、ネットワーキングおよびメンターシップ機会を提供するために設立されました。グループはまた、セキュリティ業界への参入や学習に興味のある女性や少女にインスピレーションを与えることも目指しています。

### メンバーシップ

GitLab Security 組織の現在の女性チームメンバー向けに調整されていますが、セキュリティ業界の女性のサポートに専念しているすべての GitLab メンバーの参加を奨励し歓迎します。

### プログラム

- リーダーシップとネットワーキングに合わせた AMA
- メンターシップ
- ソーシャルアウトリーチ
- 出版
