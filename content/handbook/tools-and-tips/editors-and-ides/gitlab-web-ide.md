ウェブサイト: <https://docs.gitlab.com/ee/user/project/web_ide/>

得意分野: GitLab の Web UI から直接編集

強み:

- GitLab 公式の Visual Studio Code 版で、GitLab 内の任意のプロジェクトやファイルに対して直接実行できます
- GitLab の [Workspaces](https://docs.gitlab.com/ee/user/project/remote_development/) 機能との統合も提供します。
