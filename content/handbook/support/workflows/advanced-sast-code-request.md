GitLab Advanced SAST における LGPL ライセンスコンポーネントのソースコードに関するリクエストには、Zendesk の **Advanced SAST App** を使用してください。アプリはプロセスを自動化し、法的な Issue が作成されるまでガイドします。

法的な Issue が作成されると、チケットで Dev Pulse が有効化されます（RFH として扱われます）。法的な Issue が更新されるまでチケットは on-hold のままで、その時点で法務チームから何を言うか／何をするかの指示があります。

Issue で追加のコミュニケーションが必要な場合は、`General::Waiting on RFH` マクロを使用して Dev Pulse を再有効化します。

チケット作業中に問題が発生したり質問がある場合は、チケットで Jason Colyer を CC し、その問題／質問に関する内部ノートを追加してください。
