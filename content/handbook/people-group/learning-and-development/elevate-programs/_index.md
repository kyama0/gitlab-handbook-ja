各 Elevate プログラムへは、以下のリンクから移動してください。適切なプログラムを選ぶための便利なガイドは次のとおりです:

| プログラム名 | 説明 |
| ----- | -------- |
| [Elevate](/handbook/people-group/learning-and-development/elevate-programs/elevate/) | マネージャーおよびシニアマネージャー向け。 |
| [Elevate Applied](/handbook/people-group/learning-and-development/elevate-programs/elevate-applied/) | Elevate 認定を取得した方向けの継続的学習とリソース |
| [Elevate+](/handbook/people-group/learning-and-development/elevate-programs/elevateplus/) | ディレクターおよびシニアディレクター向け |
| [ElevatePrep](/handbook/people-group/learning-and-development/elevate-programs/elevate-prep/) | ハイポテンシャル IC、ピープルマネージャーへのキャリアパスを歩んでいる方、あるいはピープルマネージャーになることに関心のある方向け。 |

Elevate に関するご質問や、いずれかのプログラムへの参加に関心がある場合は、#learninganddevelopment チャンネルでお問い合わせください。

#### Elevate フィードバックループ

Elevate プログラムを常に最先端に保つため、Talent Development チームは、Hone のワークショップから得られたインサイト、Elevate 参加者とマネージャーからのフィードバック、エンゲージメントサーベイのデータ、経営層からの直接の意見など、複数のセンシング手段を用いて継続的に設計をイテレーションしています。これらのシグナルが Elevate のコンテンツ、提供方法、コホート構成の継続的な改善を導き、GitLab の進化するニーズにスイート全体を整合させ続けます。これは、私たちの[製品センシング手段](/handbook/product/product-processes/sensing-mechanisms/)が継続的な製品改善を導くのと同じ仕組みです。
