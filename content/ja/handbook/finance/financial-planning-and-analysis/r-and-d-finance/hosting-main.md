### GCP - コミット使用割引（CUD）プロセス

GCP CUD プロセスの詳細については、ハンドブックの [GCP CUD](/handbook/engineering/infrastructure-platforms/cost-management/gcp-cud/#gcp-cud-purchase-approval-process) ページをご参照ください。

CUD 購入のリクエストは、CUD テンプレートを使用して [Finance Issue](https://gitlab.com/gitlab-com/Finance-Division/finance/-/issues/new) を作成し、Matt Dunleavy にアサインしてください。

### AWS - 節約プラン（SP）およびリザーブドインスタンス（RI）プロセス

注: このページは作成が必要です
AWS 節約プランのプロセスの詳細については、ハンドブックの [AWS Savings Plan](https://www.google.com/) ページをご参照ください。

注: このページは作成が必要です
AWS リザーブドインスタンスのプロセスの詳細については、ハンドブックの [AWS Reserved Instances](https://www.google.com/) ページをご参照ください。

購入のリクエストは、適切なテンプレート（AWS Savings Plan または AWS Reserved Instance）を使用して [Finance Issue](https://gitlab.com/gitlab-com/Finance-Division/finance/-/issues/new) を作成し、Matt Dunleavy にアサインしてください。
