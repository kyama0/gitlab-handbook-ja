## TMRG の目的

API TMRG は、すべての GitLab チームメンバーがアジアおよび太平洋諸島の文化をよりよく理解し、多様性が GitLab の戦略的目標とグローバルな成功にどのように貢献できるかを理解するための安全な場を提供します。この TMRG は排他的な性質を持たず、GitLab の誰でも参加できます。

## ミッション

「GitLab 内のアジアおよび太平洋諸島コミュニティの多様なバックグラウンドに声を与えることで、API 個人の進歩、文化的認識、採用、定着、および専門的発展を促進し、GitLab のメッセージが多様性、インクルージョン、ビロンギングのコアバリューを反映し続けるようにすること。」
